PurchAIs is an intelligent, multimodal grocery assistant designed to reduce household food waste by closing the loop between inventory tracking, consumption, and restocking.

## ðŸ’¡ The Concept
Modern fridges are smart, but they lack reasoning. PurchAIs fills this gap by acting as a "Kitchen Operating System" that visually understands what you have and logically suggests how to use it before it spoils.

## ðŸš€ Key Features
* **Multimodal Ingestion:** Simulates processing video feeds of an open fridge to identify ingredients and freshness levels.
* **Contextual Reasoning:** The "AI Chef" doesn't just list recipes; it solves the "Knapsack Problem" for your pantry, prioritizing expiring items.
* **Automated Logistics:** Real-time inventory state management. If an ingredient is used (cooked) and drops below a user-defined threshold, it is automatically added to the restock queue.
